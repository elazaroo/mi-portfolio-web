# Portafolio Web de Eneko

Este repositorio contiene el código fuente de mi sitio web personal. El proyecto está construido con HTML5 y CSS3, siguiendo un diseño minimalista y limpio en escala de grises.

## Características

- Estructura semántica HTML5
- hoja de estilos CSS personalizada
- Diseño responsive y minimalista

## Estructura del Proyecto

- `index.html`: Página principal
- `style.css`: Estilos del sitio
- `.gitignore`: Archivos ignorados por git
